# Personal Portfolio

A modern and responsive portfolio website built to showcase my frontend development skills, projects, and professional experience.

## ✨ Features
- Responsive design for all devices
- Modern UI with Tailwind CSS & DaisyUI
- Interactive project cards
- Work experience section
- Contact section
- Smooth scrolling navigation
- Animated project image slider

## 🛠️ Technologies
- HTML5
- Tailwind CSS
- JavaScript (ES6+)
- DaisyUI
- Font Awesome
